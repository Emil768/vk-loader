# VK-loader

# 📁 Клон известной социальной сети раздела «Файлы» 📁 
Приложение выполняет те же функции что и оригинальный раздел.

## 🛠 Технологии:
- ReactJS 18
- NodeJs
- Axios
- Express (Простой сервер)
- Multer (Помощник загрузки файлов на стороне сервера)
